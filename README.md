# Pepeballester.github.io
pagina web de fotografias para los Lopez de Villena
tiene una pagina inicial con poco texto, una segunda con unas fotografias y luego 8 galerias de imagenes correspondiente a cada uno de los hermanos
Todas las fotos estan en la carpeta photos y en cada pagina esta el enlace a las fotos que le corresponden.
No se ha creado un html separado para el menu que aparece en todas las hojas ya que ademas de estar en web tambien estara en local con lo que lso navegadoresno permite ejecutar ciertos comandos en local

Estoy preparando cambios para que la web no local este mas depurada (menu separado en un html, los codigos css en archivos css separados para ser reutilizados) etc
